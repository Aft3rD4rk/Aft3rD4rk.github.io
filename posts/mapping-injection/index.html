Basically, we will map an unused memory region to our process‚Äôs virtual address space called a file mapping object using the WinAPI CreateFileMapping. This memory region does not need to be backed by a file; instead, you can put whatever data you want there. Fun fact: This memory region can also be shared with other processes, unlike memory allocated with VirtualAlloc, which cannot be shared.
